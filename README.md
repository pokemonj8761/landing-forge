Landing Forge helps you build landing pages. You do not need to know how to code. You describe your page with a short prompt or a simple text file. The software turns these instructions into a finished website. You get tools for analytics, testing, and search engine ranking.

## 📝 Creating Your First Page

The software uses a simple way to create websites. You have two ways to start.

### The AI Method
1. Open the application.
2. Type a sentence about your product in the box. Example: "Make a page for a coffee shop that sells beans."
3. Click the Generate button.
4. Wait for the tool to build the layout.

### The YAML Method
1. Create a file on your computer and name it data.yaml.
2. Open this file with a standard text editor like Notepad.
3. Type the site details using simple labels for headers and text.
4. Drag this file into the application window.
5. Click Generate to build the site.

## 📈 Using Built-in Tools

Landing Forge includes extra features to help your business grow.

### Analytics
You can track how many people visit your page. Open the dashboard tab to see graphs. These numbers show you which parts of your page people read.

### A/B Testing
You can compare two versions of your page. The software shows one version to half your visitors and the second version to the other half. You can then see which version performs better.

### Search Engine Optimization
The program adds tags for search engines to your site. This helps your page appear in results for specific words. You do not need to change any settings for this.

## 📁 Managing Your Projects

You can save your work at any time.

1. Click the File menu.
2. Choose Save Project.
3. Name your file and pick a location.
4. Open this file later to adjust your landing page.

You can also export your page as a finished website file. This file contains the complete site. You can upload this to any hosting service to put your page on the internet.

## ❓ Frequently Asked Questions

### Do I need to pay for hosting?
The program makes the website files for you. You need a separate hosting service or domain name provider to put those files on the web.

### Can I change the fonts or colors?
Yes. You can change these details in the settings menu. You can pick colors that match your brand.

### Does the software work offline?
You need an internet connection to use the AI prompt feature. Once the page exists, you can edit it while offline.

### How often should I check for updates?
The software checks for new features when you start it. You can see the version number at the bottom of the main window.
